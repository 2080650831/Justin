1、项目说明
Justin.BI.*是BI所用的基本类库
Justin.Controls.*是各种工具类库
Jusint.Core、Justin.FrameWork、Justin.FrameWork.WinForm、Justin.Log、Justin.Toolbox、Justin.Workspace是够成工具箱的主题
2、目录说明
   common	通用、开源类库，
   common\Library 类库dll、ocx等
   Common\Resource 开源类库源码
   Justin.Application 自己实现的几个应用、包括工具箱、股票软件、薪水计算等
   Justin.Controls 包括各种自己写的类库，工具箱用要调用
   Justin.FrameWork 框架通用底层代码、包括日志
   Release 框架dll，即Justin.FrameWork下的通用底层项目dll生成位置， 
   Release\Controls 类库dll，即Justin.Controls生成位置
